# Chapter ② — REST APIs

> **Learning objective:** Grasp the principles of RESTful architecture — resources, URIs, statelessness, representations — and understand how REST leverages plain HTTP to build clean, predictable APIs.

> This chapter builds directly on **[Chapter ①](#01-http)**. REST is not a new technology; it's a *discipline* for using HTTP well. If methods and status codes feel shaky, go back first.
